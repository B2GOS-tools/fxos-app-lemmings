Known Issues

As stated on the main menu, this is work in progress (in fact I didn't expect it to become public so soon). There are still several bugs and issues in this version that I am aware of, so just to prevent you from reporting those to me, here's a list:

  • Fast forward isn't working correctely!
    That's because the game itself already takes up a lot of your CPU time. Whereas the original version could be played on a 7Mhz Amiga, now you'll need at least 500Mhz, but preferably more. Once the CPU is already 100% utiziled it makes no sense to use the fast forward; it makes the timer go off with the animations because the animations are harder (go slower) than the timekeeping itself.
    So if you haven't got a really really fast PC (like 3Ghz or so - preferably AMD because contrary to what they want you to believe P4's perform quite poor when it comes to DHTML) stay away from the fast forward, at least with a lot of lemmings™ around. This is a NO-FIX - there is simply not much I can do about it.
  • The nuke-button isn't working!
    Try double-clicking it ;) (safety precaution)
  • Not all lemmings™ explode when I nuke them!
    The nuke-effect itself, although pretty, is quite heavy for the browser to display. Therefor I have limited the effect to 10 lemmings™ max; else it might lock up your pc.
  • I can't use my functionkeys, my right-mouse-button or select any text!
    That's because I have disabled it; you shouldn't see this as a normal webpage but as an application. There are too many things depending upon each other that allowing normal browsercontrols could disrupt functionality. The Esc key will always bring you back to the main menu.
