Lemmings™ - Help

Requirements

Unlike the original game DHTML lemmings™ requires a lot of resources (around 10 - 20MB of memory) and a fast processor to run smoothly. A 500MHz processor will do, but above 1GHz it becomes much more enjoyable. You will need at least a 2GHz processor for the fast forward mode to work correctely (AMD's are faster than P4's :P )

Javascript isn't a fast programming language because it needs to be interpreted at execution time. However it does have the advantage that most browsers are javascript enabled, so you won't need anything more than a modern browser.

DHTML Lemmings™ has been written to work in IE5.5 and above and in recent Mozilla versions (around 1.3 and higher). Other browsers (such as Opera) may or may not be able to run DHTML Lemmings™

Note that this project is a work in progress, by no means is this version complete and free of bugs; feel free to report anything unusual to me.

You can also run lemmings™ offline, the complete sourcecode (except for elements that may be copyrighted) can be found here and I also have a PHP script to generate the JS levelcode with examples which can be found here.
